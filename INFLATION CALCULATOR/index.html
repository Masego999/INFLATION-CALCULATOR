The Mathematics Behind It
FV = PV × (1 + r)n

Where FV = future equivalent value  ·  PV = present (starting) value  ·  r = annual inflation rate  ·  n = number of years. This is the same compound growth formula used in treasury to model the time value of money, bond pricing, and cash flow forecasting.
