# INFLATION-CALCULATOR
nflation &amp; purchasing power calculator built with HTML, CSS, and JavaScript — portfolio project for TreasuryONE Graduate Intern application.
nflation Calculator
A clean, interactive web tool that calculates how inflation erodes purchasing power over time using the compound inflation formula:
FV = PV × (1 + r)ⁿ
Features

Enter any starting amount, year range, and annual inflation rate
Calculates equivalent value and total purchasing power lost
Interactive line chart showing value growth year by year
Year-by-year breakdown table
Pre-loaded with South African ZAR defaults

Tech Stack

HTML5 · CSS3 · Vanilla JavaScript
Chart.js for data visualisation

Relevance
Built to demonstrate applied mathematics and frontend development skills relevant to treasury operations — specifically the time value of money, compound growth, and financial data visualisation concepts used in cash management and treasury systems.
How to Run
Clone the repo and open inflation_calculator.html in your browser. No dependencies or installation required.
